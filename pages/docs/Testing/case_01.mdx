# TC 1.1: Eingabe von Plastik in die Simulation

### Beschreibung
Überprüfen, wie die Simulation auf die Eingabe von Plastikmüll reagiert.

### Annahmen und Voraussetzungen
Simulation ist gestartet.
Eingabeoption für Plastik ist verfügbar.

### Testdaten
Plastik als Eingabewert.

### Auszuführende Schritte
Starte die Simulation.
Wähle die Eingabeoption "Plastik".
Beobachte die Reaktion in der Simulation.

### Erwartetes Ergebnis
Der Zustand des Algen-Ökosystems verschlechtert sich, und eine Warnung wird angezeigt.
